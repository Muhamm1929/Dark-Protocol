# Dark Protocol

Сайт с 3 испытаниями, скрытой админ-панелью и учётом результатов участников.

## Запуск локально

```bash
npm install
npm run dev
```

Открой: http://localhost:3000

## Основные механики

- в уровнях 1 и 2 можно пробовать неограниченно;
- в уровне 3 доступно ровно 10 попыток;
- после победы показывается окно `[ ACCESS GRANTED ]` и поле для Telegram username;
- после провала показывается окно `[ ACCESS DENIED ]`;
- админ видит журнал участников в формате `user approved @telegram.` или `user failed` (дубликаты имён сохраняются);
- режимы системы из админ-панели:
  - `Возобновить испытание`
  - `Ожидание` (`[ SYSTEM LOADING ]`)
  - `Закончить испытание` (`[ TEST COMPLETED ]`)
- в экранах ожидания/завершения/результата есть маленькая кнопка входа в админ-панель с секретным кодом.

## Настройки админ-панели

- названия и описания уровней;
- шифр и ответ уровня 1;
- лог и ответ уровня 2;
- пароль и подсказка уровня 3;
- число попыток уровня 3 зафиксировано на 10.

Настройки, режим системы и журнал участников сохраняются в `localStorage`.
