# Dark Protocol

Сайт с 3 испытаниями, стартовым окном входа и скрытой админ-панелью.

## Запуск локально

```bash
npm install
npm run dev
```

Открой: http://localhost:3000

## Основные механики

- стартовое окно с вводом имени участника;
- строгая последовательность уровней: 1 → 2 → 3;
- модальное окно проверки с анимацией загрузки и результатом;
- скрытый вход в админ-панель через код `SecretAdmin` в первом уровне.

## Что можно менять в админ-панели

- названия уровней;
- описания испытаний;
- шифр и правильный ответ для уровня 1;
- лог и правильный ответ для уровня 2;
- пароль, подсказку и количество попыток для финального уровня.

Настройки сохраняются в `localStorage` браузера.

## Деплой на Vercel

1. Залей репозиторий в GitHub.
2. На Vercel нажми **Add New Project**.
3. Выбери этот репозиторий, фреймворк определится как **Next.js** автоматически.
4. Нажми **Deploy**.
